# Название проекта

Одностраничный сайт с играми.

## Описание
Учебный проект по курсу веб-разработки, созданный для демонстрации навыков вёрстки сайта в редакторе кода(VSC), создания адаптивного интерфейса под разные устройства и добавления интерактивности с помощью JavaScript.

## Технологии
Основной стек технологий:
*   **HTML5** — семантическая структура страницы.
*   **CSS3** — стилизация, Flexbox/Grid, адаптивность.
*   **JavaScript (ES6+)** — реализация логики и интерактивности (работа с DOM, использование функции prompt(), использование метода Math.random()).
*   **Git & GitHub** — контроль и место хранения версий проекта.

## Запуск проекта
Чтобы запустить проект локально, выполните следующие шаги:

1. **Клонируйте репозиторий:**
   ```bash

2. Перейдите в папку проекта.
3. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).
   *Рекомендуется использовать расширение **Live Server** в VS Code для корректной работы ссылок.*
