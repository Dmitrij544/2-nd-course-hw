# Название проекта

Веб-сайт с играми.

## Описание
Проект, представляющий собой одностраничный сайт с играми. Цель - создание визуального интерфейса с играми, которые позволяют пользователю вносить свои ответы, видеть ошибки. Задачи - создание блока со списком игр, реалезация 6 разных игр с использованием технологий JavaScript, реализация адаптивности сайта под различные устройства.

## Срок выполнения
Работа выполнена за 12 недель. Техчасть за 10 недель и доработка проекта за 2 недели

## Технологии
Основной стек технологий:
*   **HTML5** — семантическая структура страницы.
*   **CSS3** — стилизация, Flexbox/Grid, адаптивность.
*   **JavaScript (ES6+)** — реализация логики и интерактивности (работа с DOM, использование функции prompt(), использование метода Math.random()).
*   **Git & GitHub** — контроль и место хранения версий проекта.
